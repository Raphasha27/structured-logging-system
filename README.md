## 📦 Scaffolding & Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Raphasha27/structured-logging-system.git
   cd structured-logging-system
   ```

2. **Configure Environment variables**:
   ```bash
   cp .env.example .env
   ```

3. **Deploy using Docker**:
   ```bash
   docker compose up --build
   ```
